Wave is a text editor that's modern, approachable, yet hackable to
the core — a tool you can customize to do anything but also use
productively without ever touching a config file.

Packages

You choose from thousands of open source packages that add new
features and functionality to Wave—or build a package from scratch
and publish it for everyone else to use.

Themes

Wave comes pre-installed with four UI and eight syntax themes in
both dark and light colors. If you can't find what you're looking for,
you can also install themes created by the Wave community or
create your own.

Customization

It's easy to customize and style Wave. You can tweak the
look and feel of your UI with CSS/Less and add major features
with HTML and JavaScript. Check out the video on setting up Atom.
